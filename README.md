# Neonatal Sepsis Early Prediction using AI

This project focuses on early prediction of neonatal sepsis, especially in preterm infants, 
using vital signs signals: Heart Rate (HR) and Oxygen Saturation (SpO2).

The system is based on data preprocessing, time-series analysis, and a deep learning model (GRU) 
to detect early abnormal patterns before clinical symptoms appear.

## Project Structure
- Data_Preprocessing.ipynb : Data cleaning, normalization, and feature preparation
- GRU_GP.ipynb : Model training, validation, and evaluation

## Technologies Used
- Python
- NumPy, Pandas
- PyTorch / TensorFlow
- GRU-based Deep Learning Model
